Welcome to the Hackbright Python Drill System.

This system is designed to help you practice Python syntax and get familiar with programming terminology.
You will be presented with a statement in English directing you to write a small snippet of code, just a handful of lines. Write your code in the editor (pay attention to the highlighting system, it's meant to help you).
Our system will check your code against ours and provide you with our solution, then give you the opportunity to do another problem. It's important to understand that the code you will be asked to write does not do anything useful. The point of these drills are to improve your fluency with Python idioms, not to show you how those idioms are used to do useful work.
Links to reference material are provided on the right. These link back to the ThinkPython online book.
It's recommended that you do 10 drills a day, but we won't stop you if you're having fun.
